# Solar Website Core Setup

## Tech Stack

* Next.js (App Router)
* TypeScript
* Prisma ORM
* MongoDB
* Tailwind CSS
* ShadCN UI
* next-auth

## Features

* Responsive design
* SEO optimized pages
* Fast loading
* Clean UI for solar business

## Pages

* Home
* About
* Projects
* Blog
* Contact
* Get Quotation

## Requirements

* Use server components where possible
* API routes using Next.js route handlers
* Use Prisma for all DB operations
* Use Zod for validation
